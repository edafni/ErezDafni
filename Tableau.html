About the project

This project aims to transform raw data into insights through Tableau visualizations. The objective of this project is to answer questions and analyze data using simple dashboards.

about the data

This dataset includes 44,341 results of international football matches starting from the very first official match in 1872 up to 2023. The matches range from FIFA World Cup to FIFI Wild Cup to regular friendly matches. The matches are strictly men's full internationals and the data does not include Olympic Games or matches where at least one of the teams was the nation's B-team, U-23 or a league select team.

Scoring Goals analysis

In the dashboard below, we can answer the following questions by simply hovering and filtering:
1. Who is the all-time leading scorer?
2. Which country has scored the most goals?
3. What is the minute with the highest number of goals?
4. Who is the leading scorer for each game part (first half, second half, etc.)?
5. Which country has scored the most goals for each game part?
6. Who is the leading scorer for each minute of the game?
Additionally, there are many more questions that can be answered using this dashboard.
